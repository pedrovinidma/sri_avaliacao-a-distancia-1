# SRI - Avaliação à Distância
Projeto desenolvido por Pedro Vinícius Diogo Marthins de Araújo (polo São José) para a disciplina de Tópicos Especiais

## Descrição do texto utilizado
O texto utilizado é de um dataset da plataforma Hugging Face. O dataset tem o objetivo de verificar a parcialidade de notícias dos canais de notícias brasileiros Folha de São Paulo, Estadão e O Globo [(eduagarcia/FactNews)](https://huggingface.co/datasets/eduagarcia/FactNews). Deste dataset foram selecionadas 30 dessas notícias para se fazer o documento.csv para a análise.
